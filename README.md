# RD VNIIZhT 27.05.01-2017 - G‘ildirak juftligi o‘quv platformasi v2

Bu ZIP GitHub Pages uchun tayyor statik web-o‘quv platforma.

## v2 da tuzatilgan asosiy talab

Oldingi variantdagi kabi har bir bo‘limda bir xil 12 ta savol emas. Endi har bir bo‘limga savollar soni bo‘lim hajmi, mavzu murakkabligi va ishlab chiqarishdagi xavf darajasiga qarab berilgan.

- Kichik bo‘limlar: 30-45 savol
- O‘rtacha bo‘limlar: 60-120 savol
- Katta texnologik bo‘limlar: 150-270 savol
- Jami rejalashtirilgan savollar: 2745 ta
- Savollar 15 tadan guruhlarga bo‘lingan
- Har bir guruh alohida yakunlanadi va progress saqlanadi
- Tillar: O‘zbek lotin, Ўзбек кирилл, Русский

## Fayllar

- `index.html` - asosiy sahifa
- `style.css` - professional responsiv dizayn
- `app.js` - test, progress, til va guruhlash logikasi
- `data/training-data.js` - 32 bo‘lim, qamrov xaritasi, savollar generatori uchun ma’lumotlar

## GitHub Pagesga joylash

1. GitHub repository oching.
2. ZIP ichidagi fayllarni repository root qismiga yuklang.
3. Settings -> Pages -> Deploy from branch -> main / root ni tanlang.
4. 1-2 daqiqadan keyin sayt ochiladi.

## Keyingi kengaytirish

Platforma telejka, tormoz, avtotsepka, kuzov va boshqa vagon qismlarini alohida modul sifatida qo‘shishga tayyor.

## Muhim eslatma

Bu v2 platforma RD tuzilmasi va ishlab chiqarish mantig‘ini xodimlar uchun o‘quv formatiga aylantiradi. Amalda korxona ichki texnologik kartalari, attestatsiya talablari va aniq raqamli me’yorlar bilan birga qo‘llanishi kerak.
