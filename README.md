# G‘ildirak juftligi o‘quv-test platformasi

Bu arxiv GitHub Pages uchun tayyor statik platforma.

## Tarkib

- `index.html` - 30 ta bo‘lim ro‘yxati
- `lesson.html` - dars matni, jadvallar, nazorat kartasi
- `test.html` - har bo‘lim uchun 15 ta test
- `data/training-data.js` - asosiy kontent bazasi
- `data/training-data.json` - JSON zaxira nusxa
- `source_docs/` - foydalanuvchi yuborgan Word mavzulari

## GitHubga joylash

1. Arxivni oching.
2. Ichidagi fayllarni `gildirak_juftligi` repozitoriyingiz ildiziga nusxalang.
3. GitHubga commit qiling.
4. Settings → Pages bo‘limida `main` branch va `/root` tanlang.

## Eslatma

Rasmlar ishlatilmagan. Kontent 30 ta Word mavzu asosida dars + jadval + amaliy nazorat + test shakliga keltirildi.
